# Vedant Kaushik — Portfolio

Personal portfolio website for **Vedant Kaushik**, Senior Backend Engineer with 6+ years designing high-throughput distributed systems at Ciena and Nagarro.

**Live stack:** Next.js 14 · React Three Fiber · GSAP · Framer Motion · Tailwind CSS

---

## Features

- Fixed full-screen WebGL canvas — interactive neural-network particle system built with React Three Fiber
- GSAP ScrollTrigger — scroll position drives the Three.js camera through all sections
- Hero — staggered Framer Motion type animation with a magnetic CTA button
- Experience — sticky split layout with a detailed career timeline (Ciena → Nagarro)
- Projects — GSAP-pinned horizontal scroll with ←/→ arrow controls and progress dots
- Footer — "Let's Build Something." with social links

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 14 (App Router) |
| 3D / WebGL | Three.js, React Three Fiber, @react-three/drei |
| Animations | GSAP (ScrollTrigger), Framer Motion |
| Styling | Tailwind CSS, custom GLSL shaders |
| Fonts | Inter, Space Grotesk |

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build   # production build
npm start       # serve production build
```
